# devops-ecommerce-frontend

## Overview
This repository contains the frontend application for the DevOps final project e-commerce platform.

## Responsibilities
- Provide the React-based user interface
- Display product data to users
- Send requests to backend services

## Tech Stack
- React
- JavaScript
- npm

## Branching Strategy
This repository follows the Git Flow model:
- `main`: production-ready code
- `develop`: integration branch for ongoing development
- `feature/*`: new features
- `release/*`: release preparation
- `hotfix/*`: urgent production fixes
